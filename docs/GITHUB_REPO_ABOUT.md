# GitHub 仓库 About 设置建议

在 GitHub 仓库主页右侧 **About** 旁点击齿轮图标，可填写以下内容以符合 Public Good / EF 资助印象：

---

**Description（简短描述）**

```
A standard protocol for programmable cascading payments and atomic asset containers on EVM chains.
```

**Website（可选）**

- 可填 Gitbook / 协议文档链接
- 暂不填 App 下载或商业官网，更突出“协议/基础设施”

**Topics（标签）**

建议勾选或添加：

- `ethereum`
- `solidity`
- `erc-4337`
- `payment-protocol`
- `public-goods`

（`public-goods` 对 EF 类资助申请有帮助。）

---

保存后，仓库会以“开源协议 / 公共物品”形象展示。
